# Artificer v0.1.0

This release establishes the first standalone GitHub-publishable Artificer surface.

## Highlights

- standalone `artificer` launcher
- standalone installer independent of App Forge
- portable Linux bundle
- macOS `.app` bundle
- GitHub Actions build and release pipeline
- public install and contributor docs

## Requirements

- Ollama
- at least one local model
- Wizardry runtime
