# Glossary

This file exists because AI writing often assumes the reader already knows the vocabulary. Most beginners do not, and that is normal.

## LLM

Large language model. The text-generating engine underneath Artificer.

## Model

A specific LLM you have installed locally, usually through Ollama.

## Ollama

The local model server Artificer uses to run installed models.

## Workspace

A local project folder Artificer can work with.

## Conversation

A thread inside a workspace. It stores the running context for a task.

## Mode

A behavior profile that changes how Artificer frames and handles a task.

## Prompt

The instruction given to the model.

## Context

The extra text and state Artificer gives the model along with your prompt, such as recent messages or project information.

## Runtime Guidance

Extra instructions Artificer adds around the prompt to push the model toward better behavior.

## Plugin

A user-visible, reversible behavior patch that changes how Artificer steers the model.

## Local-First

The main model execution and app state live on your machine rather than depending primarily on a remote service.

## Assay

An internal evaluation or test used to measure how Artificer behaves.
